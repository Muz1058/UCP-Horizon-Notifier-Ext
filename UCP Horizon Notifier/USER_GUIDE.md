# UCP Horizon Notifier User Guide

## 1. Introduction

### What This Application Is
- Horizon Notifier is a browser extension for the UCP Horizon student portal.
- It helps students track course updates in one place.

### Who It Is For
- Students using the UCP Horizon portal
- Anyone who wants a faster way to monitor course updates

### Key Purpose
- To collect and show important course updates such as:
  - Announcements
  - Course outlines
  - Course materials
  - Submissions
  - Grades

## 2. Features Overview

- `Scan Current Courses`
  - Checks open course pages for updates
- `Scan All Courses`
  - Scans all courses from the dashboard
- `Unread Tracking`
  - Marks new items as unread so you can spot them quickly
- `Course Update Panel`
  - Shows updates in a side panel on the dashboard
- `Popup Summary`
  - Shows all recent updates from the extension icon
- `Course Tab Badges`
  - Shows unread counts inside course tabs
- `Auto Scan on Login`
  - Can automatically scan after you log in
- `Download Links`
  - Lets you open attachments and download files directly
- `Read / Unread Status`
  - Marks items as read when you view them
- `Notifications`
  - Alerts you when new updates are found

## 3. Getting Started

### How to Access the Application
- Open Google Chrome
- Install or load the Horizon Notifier extension
- Log in to `https://horizon.ucp.edu.pk`
- Open the UCP Horizon dashboard

### Basic Setup
- Click the extension icon in Chrome to open the popup
- Check that `Auto Scan on Login` is enabled if you want automatic scanning
- Make sure you are logged in to Horizon before scanning

## 4. How to Use

### Navigation Overview
- Use the Chrome extension icon to open the popup
- Use the Horizon dashboard to access the in-page toolbar and update panel
- Open any course page to see course-tab badges

### How to Scan for Updates
- Open the Horizon dashboard
- Click `Scan All Courses`
- Wait for the scan to complete
- Open the update panel or popup to review results

### How to View Updates
- Click the extension icon
- Choose a category such as:
  - Announcements
  - Outline
  - Materials
  - Submissions
  - Grades
- Click an item to view more details

### How to Mark Items as Read
- Open an item in the popup or panel
- Some items are marked as read automatically when viewed
- You can also use `All read` or `Mark all read` where available

### How to Clear Updates
- Open the popup or dashboard panel
- Click `Clear all updates`
- This removes saved update items from the extension view

### How to Download Files
- Open the popup or dashboard panel
- Find a material or outline item
- Click `Download`

### Searching or Filtering
- Use category tabs or filters to narrow what you see
- Switch between:
  - All
  - News
  - Outline
  - Material
  - Submission
  - Grades

## 6. Common User Flows

### How to Check New Course Updates
1. Log in to Horizon
2. Open the dashboard
3. Click `Scan All Courses`
4. Open the side panel or extension popup
5. Review unread items

### How to Read a New Announcement
1. Open the popup or dashboard panel
2. Find the announcement
3. Click `View Details`
4. Read the message
5. It may be marked as read automatically

### How to Download a Course File
1. Open the popup or panel
2. Go to `Material` or `Outline`
3. Find the file
4. Click `Download`

### How to Review Updates for One Course
1. Open the dashboard panel
2. Find the course name
3. Expand the course section
4. Review announcements, files, submissions, and grades

### How to Reset the Update List
1. Open the popup or side panel
2. Click `Clear all updates`
3. Run a fresh scan if needed

## 7. Error Handling

### Common Errors and What They Mean
- `No course page open`
  - You tried to scan current course pages, but no course tab is open
  - Fix: open a Horizon course page first
- `Navigate to the Horizon dashboard first`
  - `Scan All` needs the dashboard page
  - Fix: open the Horizon dashboard and try again
- `No courses found`
  - The dashboard did not detect any course cards
  - Fix: refresh the dashboard and try again
- `Scan completed with failed requests`
  - Some course pages could not be read during the scan
  - Fix: check your internet, refresh Horizon, and scan again
- `Extension context invalidated`
  - The extension was reloaded while the page was still open
  - Fix: reload the page after reloading the extension
- Missing or fewer items than expected
  - A scan may have been interrupted, or Horizon content may not have loaded fully
  - Fix: refresh the dashboard and run `Scan All` again

## 8. FAQ

### Do I need to keep Horizon open?
- Yes. The extension works with the Horizon website while you are logged in.

### Will it scan automatically?
- It can, if `Auto Scan on Login` is enabled.

### Why do some items disappear from unread view?
- They may have been marked as read after you viewed them.

### Can I still see read items?
- Yes. Use `Show All`.

### Why do I not see the Scan All button?
- You may not be on a Horizon page or dashboard.

### Does this change my data in Horizon?
- No. It only reads updates and helps you view them.

### Why does a file download open in a new tab?
- Some files are opened using Horizon's download link behavior.

## 9. Best Practices

- Scan from the dashboard for the most complete results
- Use `Show All` if you think something is missing
- Refresh the dashboard before scanning if Horizon looks outdated
- Keep `Auto Scan on Login` enabled for convenience
- Review unread badges regularly so you do not miss updates
- If something looks wrong, run a fresh scan instead of relying on old data

## 10. Conclusion

- Horizon Notifier helps you monitor course activity faster and more clearly
- It is best used from the Horizon dashboard after login
- Use the popup for quick checks and the dashboard panel for detailed review

### Support
- If the extension stops working:
  1. Reload the extension
  2. Refresh the Horizon page
  3. Run a new scan
- If problems continue, contact the person or team who provided the extension build
